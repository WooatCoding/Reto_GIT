# Reto GIT

Este proyecto corresponde al reto de Git para la materia **Ingeniería de Software II**.

Incluye:
- Creación del repositorio local
- Manejo de ramas (`main`, `dev`, `test`)
- Resolución de conflictos entre ramas
- Fusión de cambios con `merge`
- Subida del proyecto a GitHub

## Estructura del proyecto
- `app.py` — Archivo principal con los mensajes de las distintas ramas
- `login.py` — Archivo agregado en la rama `dev`
